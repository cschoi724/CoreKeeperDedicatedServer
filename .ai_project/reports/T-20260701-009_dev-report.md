- Steam AppID 후보는 `896660`, login 후보는 `anonymous`로 기록했다.
- 실행 후보는 `start_headless_server.bat`, `valheim_server.exe`로 기록했다.
- 데이터 경로 후보는 `%USERPROFILE%\AppData\LocalLow\IronGate\Valheim`로 기록했다.
- 백업 대상 후보는 `worlds`, `worlds_local`로 기록했다.
- 실제 게임별 구현이 필요한 기능은 모두 unsupported로 두었다.

```json
"features": {
  "worldImport": false,
  "configPatch": false,
  "gracefulStop": false,
  "healthCheck": false
}
```

## 확인 근거와 한계

- Valheim Dedicated Server의 AppID 후보 `896660`은 skeleton 초기값으로 사용했다.
- Steam Store appdetails API 확인:

```bash
curl -L --max-time 10 'https://store.steampowered.com/api/appdetails?appids=896660'
```

결과:

```json
{"896660":{"success":false}}
```

Dedicated Server 도구 앱은 Store appdetails에서 상세 조회가 막힐 수 있어, 위 결과만으로 AppID 유효성을 확정하지 않았다. Windows/SteamCMD 환경에서 `app_update 896660 validate` 또는 Steam 앱 목록 기반 재확인이 필요하다.

```bash
ruby -rjson -e 'ARGV.each { |p| JSON.parse(File.read(p)); puts "#{p}: JSON OK" }' automation/src/Games/CoreKeeper/game.json automation/src/Games/Valheim/game.json automation/config/games/corekeeper.example.json automation/config/games/valheim.example.json
ruby -rjson -e 'required=%w[gameId displayName steam paths server backup features]; Dir["automation/src/Games/**/game.json"].sort.each do |p| j=JSON.parse(File.read(p)); missing=required.reject{|k| j.key?(k)}; raise "#{p} missing #{missing.join(",")}" unless missing.empty?; raise "#{p} missing steam.appId" unless j.dig("steam","appId").to_s.size>0; raise "#{p} missing steam.login" unless j.dig("steam","login").to_s.size>0; raise "#{p} missing launchCandidates" unless j.dig("server","launchCandidates").is_a?(Array) && j.dig("server","launchCandidates").any?; raise "#{p} missing backup.targets" unless j.dig("backup","targets").is_a?(Array) && j.dig("backup","targets").any?; puts "#{j["gameId"]}: manifest contract OK"; end'
ruby -rjson -e 'matches=Dir["automation/src/Games/**/game.json"].map{|p| [p, JSON.parse(File.read(p))]}.select{|p,j| j["gameId"].casecmp("valheim").zero?}; raise "valheim manifest not found" if matches.empty?; matches.each{|p,j| puts "#{j["gameId"]} #{j["displayName"]} #{p}"}'
```
