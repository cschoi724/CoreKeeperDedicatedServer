- `SteamCmdManager.psm1`가 추가되었고 SteamCMD 설치, 실행, app update 인자 생성을 Core 함수로 제공한다.

- Core SteamCMD 구현과 install/update 스크립트에서 `1963720`, `anonymous`, `Core Keeper`, `Pugstorm` 하드코딩은 검색되지 않았다.
- 기존 `CoreKeeper.SteamCmd.psm1` 함수명은 wrapper로 유지되었다.
- `install-steamcmd.ps1`, `install-server.ps1`, `update-server.ps1`는 `-Game corekeeper` 기본값과 `SupportsShouldProcess`를 가진다.

- R4 산출물은 `DEVELOPMENT_PLAN.md`와 `REFACTORING_PLAN.md`의 SteamCMD Core 분리 방향과 일치한다.
- 실제 SteamCMD 네트워크 실행은 Task 제외 범위이며, Windows 검증 필요 사항으로 개발 보고서에 기록되어 있다.
