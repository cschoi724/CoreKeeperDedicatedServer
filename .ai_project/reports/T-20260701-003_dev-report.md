## 검증

macOS 환경에 `pwsh`가 없어 PowerShell import 실행은 수행하지 못했다. 대신 가능한 정적 검증을 수행했다.

```bash
ruby -rjson -e 'ARGV.each { |path| JSON.parse(File.read(path)); puts "#{path}: OK" }' automation/config/manager.example.json automation/config/games/corekeeper.example.json automation/src/Games/CoreKeeper/game.json
ruby -e 'text = File.read("automation/src/Core/AdapterManager.psm1"); opens = text.count("{"); closes = text.count("}"); abort("brace mismatch #{opens} != #{closes}") unless opens == closes; puts "automation/src/Core/AdapterManager.psm1: brace count OK"'
rg -n "joinCodePatterns|directConnect" automation/src/Games/CoreKeeper/game.json automation/config/games/corekeeper.example.json automation/config/manager.example.json automation/docs/DEVELOPMENT_PLAN.md automation/docs/ADAPTER_GUIDE.md
git diff --check
```

결과:

- JSON 3개 파일 파싱 통과
- `AdapterManager.psm1` 중괄호 개수 정적 확인 통과
- deprecated manifest 필드 `joinCodePatterns`, `directConnect` 검색 결과 없음
- `git diff --check` 통과

## Windows 검증 필요

Windows 또는 `pwsh`가 있는 환경에서 아래 명령을 추가 확인해야 한다.

```powershell
Import-Module .\src\Core\AdapterManager.psm1 -Force
Get-GameServerAdapter -Game corekeeper
```
