## 검증

macOS 환경에 `pwsh`가 없어 Task의 PowerShell import와 `-WhatIf` 실행 검증은 수행하지 못했다. 대신 가능한 정적 검증을 수행했다.

```bash
rg -n "1963720|anonymous|Core Keeper|Pugstorm" automation/src/Core/SteamCmdManager.psm1 automation/src/CoreKeeper.SteamCmd.psm1 automation/scripts/install-steamcmd.ps1 automation/scripts/install-server.ps1 automation/scripts/update-server.ps1
ruby -e 'ARGV.each { |p| text=File.read(p); abort("brace mismatch #{p}") unless text.count("{") == text.count("}"); abort("paren mismatch #{p}") unless text.count("(") == text.count(")"); puts "#{p}: structure OK" }' automation/src/Core/SteamCmdManager.psm1 automation/src/CoreKeeper.SteamCmd.psm1 automation/src/Core/ConfigManager.psm1 automation/scripts/install-steamcmd.ps1 automation/scripts/install-server.ps1 automation/scripts/update-server.ps1
rg -n "steamLogin|appId|force_install_dir|app_update|SupportsShouldProcess|WhatIf" automation/src/Core/SteamCmdManager.psm1 automation/src/Core/ConfigManager.psm1 automation/src/CoreKeeper.SteamCmd.psm1 automation/scripts/install-server.ps1 automation/scripts/update-server.ps1
git diff --check
```

결과:

- Core SteamCMD 구현과 install/update 스크립트에서 `1963720`, `anonymous`, `Core Keeper`, `Pugstorm` 검색 결과 없음
- PowerShell 파일 중괄호/괄호 구조 확인 통과
- `app_update` 인자는 `Settings["appId"]`, `Settings["steamLogin"]`에서 구성됨
- `git diff --check` 통과
